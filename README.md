# 💳 SpendDNA – Personal Expense Analytics Using Python

> Transforming raw banking transactions into meaningful financial insights using Python, Pandas, and NumPy.

---

## 📌 Project Overview

**SpendDNA** is a data analytics project that analyzes personal banking transactions and generates meaningful financial insights from raw transaction data.

The project performs data cleaning, preprocessing, feature engineering, transaction categorization, anomaly detection, spending analysis, and financial personality profiling to help users better understand their spending habits.

This project demonstrates an end-to-end data analytics workflow using Python.

---

## 🚀 Features

### 🧹 Data Cleaning & Preprocessing

- Handles missing values
- Cleans inconsistent date formats
- Converts currency values into numeric format
- Removes duplicate transactions
- Standardizes transaction types
